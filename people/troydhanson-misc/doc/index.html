About

These are miscellaneous examples and basic techniques for C programmers on Linux. These are all placed in the public domain.

How to use

  • you can browse the links below, or

  • download the repo: misc-master.zip

  • clone it with git: git clone https://github.com/troydhanson/misc.git

Here’s a link to this repo’s Github page and to my main page.
