This archive contains the second public release of Rich Hickey'S callback library 
and a modified version for the Borland Compiler 5.5X from Martin Zimmerman.


The second public release (9/2/96) consists of: 

callback.hpp      // the library proper
callback.cpp	  // source for comparison ops

cbsuite.cpp       // a non-exhaustive test suite

cbexam1.cpp       // examples
cbexam2.cpp
cbexam3.cpp
readme.txt	  // readme 

vc42bugs.cpp	  // FYI



The modified version consists of:

callbackBCB.hpp   // the modified library
readme2.txt       // this text
